# 104周量化开发者学习计划

> 从零开始，104周成为合格的量化开发工程师 🚀

## 📚 项目概述

这是一份详细的学习路线图，旨在帮助开发者在 **104 周（2年）** 内系统性地掌握成为量化开发工程师所需的所有技能。从 C++ 基础到金融衍生品定价，再到高频交易系统架构，每个阶段都有明确的学习目标和实战项目。

---

## 🎯 学习阶段划分

### 第一阶段：C++ 基石与量化数学（0-6 个月 | 第 1-26 周）

**核心目标**：掌握 C++ 编程基础和量化金融的数学基础

#### 1. C++ 基础模块（第 1-10 周）

- **第 1-2 周**：C++ 基础
  - 掌握变量、数据类型、控制流、函数、作用域
  - 完成所有基础语法练习（如计算器）
  - 💡 并行任务：开始 LeetCode（每周 2-3 道简单题）

- **第 3-4 周**：面向对象编程（OOP）核心
  - 深入理解类、对象、封装、构造/析构函数、this 指针
  - 项目：完成 Shape 项目，实现 Shape 基类和简单派生类

- **第 5-6 周**：OOP 高级特性
  - 精通继承（公有/私有）、多态（虚函数、v-table 原理）
  - 项目：重构 Shape 项目，使用多态计算面积

- **第 7-8 周**：I/O 与异常处理
  - 学习文件 I/O 流（`<fstream>`）的读写
  - 掌握 try-catch 块和基本异常安全
  - 项目：完成日志库的基本文件写入功能

- **第 9-10 周**：模板编程
  - 掌握函数模板和类模板的编写，理解模板特化
  - 项目：将日志库的记录函数改为模板，支持不同数据类型

#### 2. 数学基础模块（第 11-14 周）

- **第 11-12 周**：线性代数 I
  - 学习向量、矩阵、点积、叉积、矩阵乘法和转置
  - 完成教科书前几章的习题，为 Eigen 库做理论准备
  - 💡 这是"量化"的基础

- **第 13-14 周**：线性代数 II
  - 学习行列式、逆矩阵、线性方程组求解、特征值和特征向量
  - 完成线性代数的核心理论学习

#### 3. STL 与现代 C++ 模块（第 15-24 周）

- **第 15-16 周**：STL 容器
  - 精通 `std::vector` 和 `std::array`
  - 理解其内部结构和性能特性
  - 项目：使用 `std::vector` 重构之前的项目

- **第 17-18 周**：STL 关联容器
  - 精通 `std::map`、`std::set`、`std::unordered_map`
  - 理解红黑树与哈希表原理
  - 项目：使用 `std::map` 实现一个简单的词频统计器

- **第 19-20 周**：STL 算法与 RAII
  - 熟练使用 `<algorithm>`（`std::sort`、`std::find` 等）
  - 彻底掌握 RAII 范式
  - **项目：开始核心项目 P1 - 限价订单簿（LOB）**

- **第 21-22 周**：智能指针
  - 精通 `std::unique_ptr`、`std::shared_ptr`、`std::move`
  - 项目：在 LOB 中使用智能指针管理订单对象

- **第 23-24 周**：现代 C++ 与并发
  - 学习 Lambda、auto 关键字
  - 精通 `std::thread`、`std::mutex`、`std::lock_guard`
  - 项目：为 LOB 添加线程安全保护

#### 4. 概率论基础（第 25-26 周）

- **第 25-26 周**：概率论与统计 I
  - 学习概率分布（正态、均匀）、期望、方差、中心极限定理
  - 为 Boost.Random 和蒙特卡洛方法做理论准备

---

### 第二阶段：量化建模与工具（6-12 个月 | 第 27-52 周）

**核心目标**：学习金融理论和常用的量化库，实现基础定价模型

#### 1. 金融理论模块（第 27-34 周）

- **第 27-30 周**：金融理论 I
  - 精读《期权、期货及其他衍生品》（John Hull，前 1/3）
  - 目标：理解远期、期货、互换、期权基本概念、无套利定价
  - 💡 这是"金融"的基础

- **第 31-34 周**：金融理论 II
  - 精读《期权、期货及其他衍生品》（中 1/3）
  - 目标：精通二叉树模型、Black-Scholes-Merton 模型、Greeks

#### 2. C++ 库应用模块（第 35-42 周）

- **第 35-36 周**：Boost C++ 库 - Random
  - 学习并实践 Boost.Random
  - 生成符合特定概率分布的随机数
  - 练习为蒙特卡洛方法生成正态分布随机数

- **第 37-38 周**：Boost C++ 库 - 高级工具
  - 学习并实践 Boost.Math Toolkit 和 Boost.Date-Time
  - 练习处理金融合约中复杂的日期计算

- **第 39-40 周**：Eigen 库 I
  - 掌握 Eigen 核心数据结构（Matrix、Vector）的创建、初始化和基本代数运算
  - 练习矩阵乘法、求逆、转置等操作

- **第 41-42 周**：Eigen 库 II
  - 学习使用 Eigen 求解线性方程组和进行特征值分解
  - 项目：使用 Eigen 实现最小二乘法线性回归

#### 3. QuantLib 与定价引擎模块（第 43-52 周）

- **第 43-44 周**：QuantLib 入门
  - 成功编译和配置 QuantLib
  - 熟悉 Date、Calendar、DayCounter
  - 项目：使用 QuantLib 构建收益率曲线

- **第 45-46 周**：QuantLib 定价
  - 学习使用 QuantLib 内置的 Black-Scholes 引擎为欧式期权定价
  - 项目：验证 QuantLib 定价结果，与 Hull 书中的示例对比

- **第 47-48 周**：**P2 核心项目 - 二叉树定价器**
  - 从零实现二叉树定价器（美式和欧式）
  - 里程碑：定价结果与 QuantLib/Hull 对比验证

- **第 49-50 周**：**P2 核心项目 - 蒙特卡洛模拟**
  - 从零实现蒙特卡洛模拟引擎（欧式）
  - 里程碑：定价结果与 QuantLib/Hull 对比验证

- **第 51-52 周**：**P2 核心项目 - 有限差分法（FDM）**
  - 从零实现显式有限差分法求解器
  - 里程碑：使用 Eigen 进行矩阵运算，并验证 FDM 结果

---

### 第三阶段：系统架构与工程（12-18 个月 | 第 53-78 周）

**核心目标**：深入学习设计模式，实现完整的回测引擎系统

#### 1. 设计模式模块（第 53-56 周）

- **第 53-54 周**：设计模式 I
  - 深入学习策略模式、观察者模式
  - 思考：如何在回测引擎中应用这两个模式？

- **第 55-56 周**：设计模式 II
  - 深入学习工厂模式、单例模式
  - 思考：如何在回测引擎中应用这两个模式？

#### 2. **P3 核心项目 - 完整的回测引擎**（第 57-78 周）

- **第 57-60 周**：回测引擎 - 架构设计
  - 设计回测引擎的整体架构
  - 实现核心的线程安全事件队列
  - 里程碑：完成事件队列和核心事件类型定义（Market、Signal、Order、Fill）

- **第 61-64 周**：回测引擎 - 数据模块
  - 实现 DataHandler 模块
  - 能从 CSV 文件加载历史数据并生成 MarketEvent
  - 里程碑：引擎能够成功读取数据并填充事件队列

- **第 65-68 周**：回测引擎 - 策略模块
  - 实现抽象的 Strategy 基类
  - 实现一个具体的策略（如移动平均线交叉）
  - 里程碑：策略能够根据 MarketEvent 生成 SignalEvent

- **第 69-72 周**：回测引擎 - 投资组合模块
  - 实现 Portfolio 模块
  - 跟踪头寸、现金、计算 P&L
  - 里程碑：Portfolio 能根据 FillEvent 正确更新持仓和盈亏

- **第 73-76 周**：回测引擎 - 执行模块
  - 实现 ExecutionHandler
  - 模拟订单撮合逻辑（可简化）
  - 里程碑：能处理 OrderEvent 并生成 FillEvent

- **第 77-78 周**：回测引擎 - 集成与报告
  - 集成所有模块，运行一次完整的端到端回测
  - 生成基础报告（夏普比率、最大回撤）
  - 🎯 第四阶段最终项目：完成回测引擎

---

### 第四阶段：专业化与面试准备（18-24 个月 | 第 79-104 周）

**核心目标**：专业方向深化学习，准备顶级公司面试

#### 1. 专业方向模块

- **第 79-82 周**：专业化方向 - HFT/低延迟
  - 学习 Linux 底层、网络编程（Asio）、C++ 性能剖析（gprof、perf）
  - 学习使用工具定位性能 bottleneck
  - 💡 HFT 方向示例

- **第 83-86 周**：专业化方向 - 高级并发
  - 深入学习 `std::future`、`std::promise`、`std::atomic`、C++ 内存模型
  - 理解数据竞争和内存顺序

- **第 87-90 周**：C++ 底层与模板元编程（TMP）
  - 复习虚函数表、内存模型、模板元编程基本思想
  - 实现编译期阶乘，理解表达式模板

#### 2. 量化面试准备模块（第 91-98 周）

- **第 91-94 周**：量化面试 - 概率论
  - 系统性解决《A Practical Guide to Quantitative Finance Interviews》（绿皮书）上的概率谜题
  - 能够从第一性原理推导出谜题答案

- **第 95-98 周**：量化面试 - 逻辑与算法
  - 解决 QuantNet 上的逻辑谜题
  - 💡 并行任务：LeetCode 冲刺（中/困难难度）

#### 3. 最终冲刺模块（第 99-104 周）

- **第 99-101 周**：算法冲刺
  - 集中练习 LeetCode 中等和困难难度的题目（树、堆、动态规划、图）
  - 能够模拟面试场景，清晰地口述解题思路

- **第 102-104 周**：作品集润色与模拟面试
  - 清理和重构 GitHub 项目代码（LOB、定价器、回测引擎）
  - 编写高质量的 README
  - 与同行或通过平台进行至少 3-5 次模拟面试
